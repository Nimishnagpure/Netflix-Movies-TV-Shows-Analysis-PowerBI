🎬 Netflix Content Analysis using Python and Power BI

An end-to-end data analytics project that explores Netflix's content library using Python for data preprocessing and Power BI for interactive data visualization. This project analyzes movies and TV shows to uncover insights into genres, IMDb ratings, languages, release trends, Netflix Originals, and country-wise content distribution.

📌 Project Overview

The goal of this project is to analyze Netflix's content library and build an interactive dashboard that enables users to explore trends and key performance indicators (KPIs). The project demonstrates the complete data analytics workflow, from data cleaning and exploratory data analysis (EDA) to dashboard development.

🎯 Objectives
Analyze Netflix's content library.
Perform data cleaning and preprocessing using Python.
Conduct exploratory data analysis (EDA).
Create interactive visualizations in Power BI.
Identify trends in genres, languages, ratings, and release years.
Build dynamic KPIs and filters for user interaction.

📌 Key Insights
The dataset contains approximately 1,040 titles.
Movies account for the largest share of Netflix's catalog.
English is the most common language.
Netflix Originals represent a significant portion of the content.
The average content duration is approximately 90 minutes.
Most content has been released after 2010.
The United States contributes the highest number of titles.
Sci-Fi is among the most represented genres.
IMDb ratings range from 0.5 to 10.0.
